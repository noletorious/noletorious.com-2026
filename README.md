# noletorious.com

A minimalist, high-contrast developer portfolio built with Vite, React, TypeScript, Tailwind CSS 4, and shadcn/ui.

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- Vite + React (TypeScript)
- Tailwind CSS 4.0
- shadcn/ui (Radix Primitives)
- Motion.dev (animations)
- Sanity.io (CMS)
